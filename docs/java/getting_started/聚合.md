# 聚合(前台)服务

### 依赖

```xml
<parent>
    <groupId>com.chinasofti.futurelab</groupId>
    <artifactId>futurelab-base</artifactId>
    <version>3.0-SNAPSHOT</version>
</parent>

<dependencies>
    <dependency>
        <groupId>com.chinasofti.futurelab</groupId>
        <artifactId>web-mvc</artifactId>
        <version>${futurelab-base.version}</version>
    </dependency>
</dependencies>

<build>
    <plugins>
        <plugin>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-maven-plugin</artifactId>
        </plugin>
    </plugins>
</build>
```

### bootstrap.yml

```yaml
server:
  port: {端口号}
spring:
  application:
    name: {服务名}
  cloud:
    config:
      failFast: true
      profile: local
      name: ${spring.application.name}
      username: admin #配置中心密码
      password: admin
      uri:
        - {configserver地址1} #例如http://localhost:7011/ #请查看`java管理群的公告`
        - {configserver地址2}
```

### application.yml

```yaml
app:
  gateway:
    uri: localhost:9787
```

### main

```java
@SpringBootApplication
public class Application {

    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}
```

## 功能说明

与中台服务相比,前台服务没有服务发现


## 生产要求

聚合服务在uat和prod环境中必须配置

```yaml
app.web.project: true
```
