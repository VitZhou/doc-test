# 流水线

目前利用软开云流水线实现了CI、CD功能。

## 环境与分支对应关系

根据团队规范，`sit` 分支对应 `sit` 环境。

`uat` 和 `prod` 环境对应`master` 分支。

不同环境对应不同的流水线。

## 自动触发

需要在`sit`环境验证功能时，直接将 `dev` 合并到 `sit` 环境即可自动触发流水线。

目前只有 `sit` 支持自动触发。

## 手动触发

下面以测评项目举例。

1、 登录软开云 devcloud

2、 鼠标hover到对应项目上，点击流水线。
![项目](project-list.png)

3、 进入流水线列表，选择对应前端流水线任务执行。
所有前端流水线命名规律均为：

`xxx环境`+`xxx项目`+UI流水线

![流水线](pipeline-name.png)

4、右侧点击执行。
